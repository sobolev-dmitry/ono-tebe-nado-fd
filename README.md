# Проектная работа «Оно тебе надо»

Веб‑страница аукциона необычных вещей — проекта, в который никто не верил. Страница включает шапку с навигацией, обложку с заголовком и кнопкой, секцию с карточками лотов, блок с информацией об аукционе и подвал с контактами и ссылками.

## 📸 Скриншот проекта

![Интерфейс аукциона «Оно тебе надо»](/images/screen-1.png)

*Главная страница: шапка с меню и адресом, обложка с заголовком «Оно тебе надо», секция с карточками лотов, информация об аукционе и подвал сайта.*

## 🎯 Цель проекта

Отработать навыки вёрстки по макету: создать семантически правильную HTML‑структуру и стилизовать страницу с использованием CSS (гриды, флексы, позиционирование).

## ✅ Реализованные блоки

- **Шапка (header)**: меню, логотип, блок с адресом; выравнивание элементов через CSS Grid.
- **Обложка (cover)**: заголовок с переносом строк, подзаголовок, кнопка «Сделать ставку», фоновое изображение и оверлей.
- **Секция с карточками (lots)**: сетка карточек лотов с фоновыми изображениями, оверлеями и контентом; адаптация при добавлении новых карточек.
- **Секция «Об аукционе» (about)**: две колонки (логотип и текст), выравнивание через CSS Grid.
- **Подвал (footer)**: адрес, меню навигации, ссылки на соцсети; расположение элементов через гриды.

## 🛠️ Использованные технологии

- **HTML5** — семантическая разметка страницы (header, main, footer, article, address и др.).
- **CSS3** — стилизация и интерактивность:
  - CSS Grid для сложных макетов (шапка, секция «Об аукционе», подвал);
  - Flexbox для списков (меню, карточки, колонки текста);
  - абсолютное позиционирование для оверлеев;
  - фоновые изображения (background-size: cover, background-position: center);
  - кастомные кнопки и стилизация ссылок.
- **Методология прогрессивного джипега** — поэтапная детализация: от общей структуры к стилизации отдельных элементов.

## 📝 Примечания

- Вёрстка выполнена по макету с соблюдением Pixel Perfect (с помощью плагина и инструментов разработчика).
- Использованы переиспользуемые компоненты (оверлей, блок адреса).
- Шрифты и изображения подключены из заготовки проекта.
- Стили организованы в отдельных CSS‑файлах: глобальные (`global.css`) и локальные (для текущей страницы).
